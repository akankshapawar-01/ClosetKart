# ClosetKart 🛍️

A full-stack MERN Ecommerce application for trendy clothes for Men, Women & Kids.

## 🚀 Live Demo
Coming soon...

## 🛠️ Tech Stack

**Frontend:**
- React.js
- Bootstrap 5
- React Router DOM

**Backend:**
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication
- Bcryptjs

## ✨ Features

- User Registration & Login
- JWT Authentication
- Product Listing with Category Filter (Men, Women, Kids)
- Product Details Page
- Add to Cart
- Place Order with Shipping Address
- Admin Panel — Add & Delete Products
- User Profile Page
- Responsive Design

## ⚙️ Installation

### Backend:
```bash
cd backend
npm install
npm run dev
```

### Frontend:
```bash
cd frontend
npm install
npm run dev
```

## 🔑 Environment Variables

Create `.env` file in backend folder:
```
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_jwt_secret
```
